Find an app, keep your data

Solid apps read and write to a personal data store that you own, so you can switch apps without leaving your information behind.

★ Editor’s picks

A short list of polished, general-purpose apps to try first.

    Jump to a category

    • The ODI Solid File Manager★

      Browse, upload, and organise the files and folders in your Pod from a friendly web interface.

      Source code on GitHub (solid/solid-file-manager).

    • Media Kraken★

      Track the films you have watched and want to watch, and build your own collections.

    • Penny★

      A developer-friendly Pod browser for inspecting and editing the raw data in your Pod.

    • Umai

      Save and cook your recipes offline; they sync back to your Pod when you are online.

    • 0data Hello with Solid

      A minimal demo to-do app showing how a Solid client reads and writes tasks on your Pod.

    • Booklice

      Save bookmarks to your Pod and share curated lists with the community.

    • dokieli

      Write, publish, and annotate articles in the browser, with decentralised social interactions built in.

    • Pod Pro

      A developer IDE for editing the raw data in your Solid Pod.

    • PodOS Browser

      Browse and manage the resources inside your Pod through a graphical, concept-aware interface.

    • Solid File Manager

      Manage the files in your Pod through a drag-and-drop web interface.

    • Solidflix

      Track films, share watchlists with friends, and get personalised recommendations, all stored in your Pod.

    • NotePod

      Take encrypted notes in your Pod and share them selectively with people you choose.

      Desktop build on GitHub (anusii/notepod).

    • HealthPod

      Keep your health records encrypted in your Pod and share them with clinicians on your terms.

      Desktop build on GitHub (anusii/healthpod).

    • MovieStar

      Track what you watch, rate and share it, and get recommendations, with optional encryption in your Pod.

      Desktop build on GitHub (anusii/moviestar).

    • SolidFocus

      Manage tasks and Markdown notes across workspaces, online or offline, with everything saved to your Pod.

    • Portable LibreChat

      An open-source AI chat client for providers like Anthropic, OpenAI, and Azure that stores your conversation history in your Pod.

      Upstream project: librechat.ai. Source code on GitHub (solid/LibreChat).

    • Tired Bike

      Connect with hosts who can put you up for the night while you are touring by bicycle.

    • PodCal

      Share Calendly-style booking pages backed by your Pod, with CalDAV sync and team scheduling.

    • SolidOS Databrowser

      Browse and edit Linked Data in any Solid Pod with the original SolidOS data browser (Mashlib).

      Source code on GitHub (SolidOS/mashlib).

    • Synara AI

      A privacy-first AI assistant that saves a searchable log of your conversations to your own Pod.

    • Pod Drive

      Upload and organise files in your Pod through a Google-Drive-style interface.

    • Pod Resume

      Build a résumé with AI assistance; your CV and work history stay in your own Pod.

    • HomeVault

      Catalogue your belongings for insurance and household records, all saved privately to your Pod.

    • VitalStats

      Log blood pressure, weight, and other vitals privately to your own Pod.

    • InnerPod

      A guided meditation timer that saves each session to your Pod, with optional encryption.

      Source code on GitHub (gjwgit/innerpod).

    • MSFatigue

      A multiple-sclerosis fatigue survey that stores encrypted responses in the participant’s Pod, with opt-in sharing to researchers.

      Source code on GitHub (anusii/msfatigue).

    • FilePod

      Browse, upload, and download files on your Pod from a cross-platform Flutter app.

      Source code on GitHub (anusii/filepod).

    • KeyPod

      An encrypted password manager backed by your Pod; also serves as a Flutter template for Solid developers.

      Source code on GitHub (anusii/keypod).

    • GeoPod

      Save places and GPS tracks to an encrypted map stored in your own Pod.

      Source code on GitHub (gjwgit/geopod).

    • SecureDialog

      Log diabetes readings privately; entries are encrypted and stored in your own Pod.

    • TodoPod

      Manage your tasks with every entry encrypted and stored privately in your own Pod.

      Built by Togaware. Source code on GitHub (gjwgit/todopod).

    • RoloPod

      Keep your address book in your Pod, share contacts selectively, and dial them from the Android and iOS apps.

      Built by Togaware. Source code on GitHub (gjwgit/rolopod).

    • KonaPod

      Collect and analyse your Hyundai vehicle data in your Pod, sharing selected parts on your own terms.

      Built by Togaware. Source code on GitHub (gjwgit/konapod).

    • BilliPod

      Track upcoming, scheduled, and past bills; reminders are drawn from your payment history so you never miss one.

      Built by Togaware. Source code on GitHub (gjwgit/billipod).
