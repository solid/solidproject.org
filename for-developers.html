Solid is a file system for the Web. In the same way that you can store a PDF on your laptop and then choose to read or edit it using one of a dozen other applications, Solid lets you put documents into an online file system. You have control over which people, websites, apps, and AI agents can read and edit them, and when they can do so.

If you’re interested in developing an app with Solid, you can find technical documents, tutorials, and pod providers here.
